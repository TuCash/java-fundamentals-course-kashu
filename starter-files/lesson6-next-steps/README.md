# Lesson 6 — Next steps

This folder contains a small example that demonstrates:

- Handling command-line arguments.
- Simple input validation (parsing integers).

Files:

- `Main.java` — accepts commands like `greet` and `sum <a> <b>`.

How to compile and run (PowerShell / Windows):

```powershell
javac Main.java
java Main greet
java Main sum 2 3
```

Expected output samples:

```
Lesson 6 — Next steps: small utilities and argument handling

Hello! This is an example of how to pass arguments to the program.

-- and --

2 + 3 = 5
```
