# Recursos del Curso de Java

Este directorio contiene recursos adicionales para el curso de Java Fundamentals.

## 📑 Contenido

### Diagramas
- Diagramas de flujo para estructuras de control
- Diagramas de clases para ejemplos de POO
- Visualizaciones de conceptos clave

### Presentaciones
- Slides complementarias para cada lección
- Material de repaso
- Ejercicios adicionales

### Referencias Rápidas
- Cheat sheets de sintaxis Java
- Mejores prácticas y patrones comunes
- Solución de problemas frecuentes

## 🎯 Ejercicios Adicionales

Para cada lección del curso, encontrarás ejercicios adicionales en la carpeta correspondiente:

- `/lesson1-exercises`: Práctica con salida por consola y comentarios
- `/lesson2-exercises`: Ejercicios de variables y operadores
- `/lesson3-exercises`: Desafíos de control de flujo
- `/lesson4-exercises`: Práctica con métodos
- `/lesson5-exercises`: Ejercicios de POO
- `/lesson6-exercises`: Práctica con excepciones
